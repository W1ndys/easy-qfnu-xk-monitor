## 选课轮次接口

### 1. 获取选课轮次列表

**URL：**
```
GET http://zhjw.qfnu.edu.cn/jsxsd/xsxk/xklc_list
```

**请求方式：** GET

**请求头：** 需携带登录后的 Cookie（JSESSIONID）

---

### 2. 进入选课轮次

**URL：**
```
GET http://zhjw.qfnu.edu.cn/jsxsd/xsxk/xsxk_index?jx0502zbid={jx0502zbid}
```

**请求参数：**

| 参数名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| `jx0502zbid` | string | 是 | 选课轮次ID |

---

## 选课轮次数据结构

```go
type SelectionRound struct {
    ID   string // jx0502zbid - 轮次唯一标识
    Name string // 轮次名称，如"正选"、"补退选"等
    URL  string // 完整的选课入口URL
```

## 页面解析逻辑

选课轮次页面是 **HTML 页面**，不是 JSON 接口，需要通过解析 DOM 获取数据：

### 解析步骤：

1. **获取"进入选课"链接**
   - 优先从 `#jrxk` 元素的 `href` 属性获取
   - 如果 `href` 为空或是 `javascript:`，则从 `onclick` 属性提取
   - `onclick` 格式：`jrxk('jx0502zbid值')`

2. **获取所有可选轮次列表**
   - 查找所有包含 `onclick*='xsxkFun'` 的 `<a>` 标签
   - 从 `href` 属性解析 URL 参数 `jx0502zbid`
   - 从标签文本获取轮次名称

### 示例 HTML 结构：

```html
<!-- 进入选课按钮 -->
<a id="jrxk" href="/jsxsd/xsxk/xklc_view?jx0502zbid=xxx">进入选课</a>
<!-- 或 -->
<a id="jrxk" onclick="jrxk('xxx')" href="javascript:void(0)">进入选课</a>

<!-- 可选轮次列表 -->
<a href="/jsxsd/xsxk/xklc_view?jx0502zbid=xxx" onclick="xsxkFun()">正选</a>
<a href="/jsxsd/xsxk/xklc_view?jx0502zbid=yyy" onclick="xsxkFun()">补退选第1次</a>
```

## 数据库批次管理 API（后端内部）

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 获取批次列表 | GET | `/api/admin/batches?semester={semester}` | 按学期查询批次 |
| 创建批次 | POST | `/api/admin/batches` | 创建新批次 |
| 更新批次 | PUT | `/api/admin/batches/:id` | 更新批次信息 |
| 删除批次 | DELETE | `/api/admin/batches/:id` | 删除批次 |

### 批次请求/响应格式

**创建批次请求：**
```json
{
  "semester": "2024-2025-1",
  "name": "正选"
}
```

**获取批次列表响应：**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "semester": "2024-2025-1",
        "name": "正选",
        "created_at": "2024-01-01T00:00:00Z"
      }
    ]
  }
}
```

## 流程说明

1. **登录后** → 访问 `xklc_list` 页面获取选课轮次列表
2. **解析轮次** → 提取所有可选的 `jx0502zbid` 和轮次名称
3. **进入轮次** → 访问 `xsxk_index?jx0502zbid=xxx` 进入具体轮次
4. **开始选课** → 进入后即可使用五个搜索模块进行选课操作
